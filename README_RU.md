# ESRConstructor6

## Общая информация

Конструктор, упрощающий построение простых отчётов с использованием Apache POI. 
Библиотека скомпилирована на Java 1.6 и использует Apache POI 3.17.

## Общий порядок создания отчёта
1. Создать новый конструктор отчёта ESRConstructor5, в который необходимо передать созданную книгу Workbook. Могут быть использованы HSSFWorkbook, XSSFWorkbook, SXSSFWorkbook.
2. Создать отчёт ESRReport. Отчёт может содержать в себе предварительный информационный блок ESRTitleBlock, Блок заголовков ESRHeaderBlock и множество информационных блоков ESRTableBlock, в которых находится информация отчёта. Обратите внимание, что в одном конструкторе можно комбинировать множество отчётов. Каждый отчёт будет начинаться на новом листе относительно предыдущего.
3. Для добавления информации в начале отчёта (например, строка с описанием отчёта и какая-то другая вводная информация) используется предварительный информационный блок ESRTitleBlock. Данный блок может содержать множество строк. Данный блок всегда будет располагаться в самом начале отчёта. Данный блок всегда будет объединять максимальное количество столбцов, используемых в отчёте. Блок может содержать множество текстовых строк, которые добавляются при помощи метода addTitleRow(String titleRow). Блоку можно добавить стиль CellStyle.
